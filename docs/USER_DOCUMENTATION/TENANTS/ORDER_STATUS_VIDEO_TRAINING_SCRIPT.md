- Symptoms: Klik action tapi status tidak berubah

  1. Check internet connection

  3. Verify user permissions
  4. Contact support jika masih bermasalah

- Symptoms: Klik stage tapi modal tidak muncul
- Solutions:
  1. Clear browser cache
  2. Disable ad blockers sementara
  3. Coba browser lain
  4. Pastikan JavaScript enabled

**Issue 3: Actions Not Available**
- Symptoms: Tombol yang diharapkan tidak ada
- Solutions:
  1. Check user role dan permissions
  2. Verify order dalam status yang benar

  4. Contact admin untuk permission update"

## üé¨ Production Guidelines

### **Technical Requirements**
- **Resolution**: 1920x1080 (Full HD)
- **Frame Rate**: 30 fps
- **Audio**: Clear voiceover, no background music
- **Format**: MP4 with H.264 encoding
- **Subtitles**: Indonesian and English subtitles

### **Visual Guidelines**
- **Cursor**: Use large, visible cursor
- **Highlights**: Use colored boxes/arrows for emphasis
- **Zoom**: Zoom in on important details
- **Transitions**: Smooth transitions between sections
- **Annotations**: Clear text overlays for key points

### **Audio Guidelines**
- **Voice**: Clear, professional Indonesian voice
- **Pace**: Moderate pace, allow time for viewers to follow
- **Pronunciation**: Clear pronunciation of technical terms
- **Volume**: Consistent audio levels throughout

### **Post-Production**
- **Editing**: Remove long pauses, add smooth transitions
- **Captions**: Add accurate captions in both languages
- **Chapters**: Create chapter markers for easy navigation
- **Thumbnails**: Create attractive thumbnails for each video

### **Distribution**
- **Platform**: Internal training portal
- **Access**: Restricted to admin users
- **Updates**: Version control for updated content
- **Feedback**: Collect viewer feedback for improvements
