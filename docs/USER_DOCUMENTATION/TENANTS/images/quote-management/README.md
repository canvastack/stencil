# Quote Management Screenshots

This directory contains screenshots for the Quote Management Guide documentation.

## Directory Structure

- `specifications/` - Screenshots showing product specifications display
- `calculations/` - Screenshots showing pricing calculations breakdown
- `complete-views/` - Screenshots showing complete quote form views
- `comparison/` - Before/after comparison screenshots

## Required Screenshots

See `QUOTE_MANAGEMENT_SCREENSHOTS_GUIDE.md` in the parent directory for detailed specifications on what screenshots to capture and how to add them to the documentation.

## Guidelines

- Use consistent test data across all screenshots
- Optimize images for web (max 500KB per image)
- Use PNG format for static images
- Use GIF or MP4 for animations
- Capture at 2x resolution for retina displays
